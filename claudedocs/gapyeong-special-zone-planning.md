# 가평 특구 기사 시각화 기획안

## 페이지 구조 설계

### 1. Hero Section
**목적**: 핵심 의제를 강렬하게 제시

```
┌─────────────────────────────────────┐
│  [공직감시 뱃지]                      │
│                                     │
│  통일교 타운 성지화…                 │
│  '특구' 지정까지 추진                │
│                                     │
│  가평군이 통일교의 '성지화' 사업에    │
│  공적 자원을 투입하려 한다           │
│                                     │
│  [날짜] [기자]                       │
└─────────────────────────────────────┘
```

**구성 요소**:
- 카테고리 뱃지: "공직감시" (orange/yellow 계열)
- 제목: 2단 구성, 주요 키워드 강조
- 부제: 핵심 의제 한 문장 요약
- 메타정보: 날짜, 기자명

### 2. 핵심 의문 (Core Question)
**목적**: 기사의 핵심 질문 제시

```
┌─────────────────────────────────────┐
│  ⚠️ 핵심 의문                        │
│                                     │
│  "왜 통일교의 계획과 가평군의 계획이  │
│   100% 일치하는가?"                  │
│                                     │
│  뉴스타파는 통일교 내부 문건과        │
│  가평군 사업 계획안을 입수했다.       │
│  두 계획은 정확히 일치했다.           │
└─────────────────────────────────────┘
```

**스타일**:
- GlassCard with red accent
- AlertTriangle icon
- Bold question text

### 3. 한학자 총재의 3가지 지시
**목적**: 통일교의 의도를 명확히 전달

```
┌──────────────┬──────────────┬──────────────┐
│  지시 ①      │  지시 ②      │  지시 ③      │
│  🏛️ 바티칸   │  🚡 관광명소 │  🌳 에덴동산 │
│              │              │              │
│  "천원단지를 │ "관광객      │ "에덴동산을  │
│   바티칸같은 │  1천만명"    │  만들어야"   │
│   성지로"    │              │              │
│              │              │              │
│  [세부내용]  │  [케이블카]  │  [클러스터]  │
└──────────────┴──────────────┴──────────────┘
```

**구성**:
- 3 column grid (모바일: stack)
- 각 카드에 대표 아이콘
- 인용문 큰 글씨로 강조
- 세부 실행 계획 하단 표시

### 4. 사업 비교 - Split View
**목적**: 계획의 일치를 시각적으로 증명

```
┌─────────────────────────────────────┐
│      사업 계획 비교: 완벽한 일치      │
├──────────────┬──────────────────────┤
│              │                      │
│  통일교 계획  │    가평군 계획       │
│  (내부문건)  │    (공식문서)        │
│              │                      │
│  케이블카    │ ←──100%일치──→ 케이블카 │
│  • 출발지    │                • 출발지│
│  • 경로      │                • 경로 │
│  • 종착지    │                • 종착지│
│              │                      │
│  에덴동산    │ ←──100%일치──→ 관광   │
│  • 시작점    │           클러스터    │
│  • 범위      │                • 시작점│
│  • 종료점    │                • 범위 │
│              │                • 종료점│
└──────────────┴──────────────────────┘
```

**인터랙션**:
- Hover시 연결선 애니메이션
- 일치하는 항목 하이라이트
- 토글로 지도 오버레이 표시 가능

### 5. 기회발전특구 메커니즘
**목적**: 공적 자원 흐름 시각화

```
┌─────────────────────────────────────┐

│                                     │
│      [가평군]                        │
│         ↓                           │
│    특구 지정                         │
│         ↓                           │
│  ┌──────┴──────┐                   │
│  │             │                   │
│  재정지원    규제완화                │
│  │             │                   │
│  └──────┬──────┘                   │
│         ↓                           │
│   [수혜 대상]                        │
│   • HJ디오션리조트 (통일교)          │
│   • 아이티씨코리아 (통일교)          │
└─────────────────────────────────────┘
```

**스타일**:
- 플로우차트 형태
- 화살표 애니메이션
- 수혜 대상 강조 (통일교 연결)

## 애니메이션

1. **Hero**: Fade in with slight scale
2. **Cards**: Stagger animation (0.1s delay each)
3. **Split View**: Slide from sides + connection line draw
4. **Flow Chart**: Sequential reveal (top to bottom)
5. **Scroll Progress**: Top progress bar

## 반응형 브레이크포인트

```
Mobile:  < 768px  - Stack all cards
Tablet:  768-1024px - 2 column grid
Desktop: > 1024px - 3 column grid
```

## 상호작용 요소

1. **Hover Effects**:
   - Cards: border glow + slight lift
   - Icons: color change + scale
   - Connection lines: pulse animation

2. **Click Interactions**:
   - 증거 문건: Expand for full text
   - 지도: Toggle overlay view
   - 비교표: Highlight matching items

## 성능 최적화

- Framer Motion: Lazy load below fold
- Images: WebP format with fallback
- Animations: Reduce motion preference respect
- Code splitting: Dynamic imports for heavy components

## 접근성

- ARIA labels for all interactive elements
- Keyboard navigation support
- High contrast mode support
- Screen reader optimized structure
- Focus indicators
