# 김병기 배우자 비리 은폐 기사 시각화 기획안

## 페이지 구조 설계

### 1. Hero Section
**목적**: 이중 거짓말의 충격을 전달

```
┌─────────────────────────────────────┐
│  [공직감시 뱃지]                     │
│                                     │
│  "이재명이가"...                    │
│  '배우자 비리 은폐' 김병기,          │
│  민주당·국민에 거짓말                │
│                                     │
│  당 지도부와 국민,                   │
│  모두를 속인 이중 기만               │
│                                     │
│  [날짜] [기자]                       │
└─────────────────────────────────────┘
```

**구성 요소**:
- 카테고리 뱃지: "공직감시" (노란색/경고)
- 제목: 육성 인용 ("이재명이가")
- 부제: 이중 기만 강조

## 상호작용

1. **Timeline**: Click으로 각 시점 상세 보기
2. **Split View**: Hover로 각 거짓말 결과 강조
3. **Voice Quotes**: Expand로 전체 맥락 보기
4. **Flowchart**: 각 단계 클릭으로 증거 자료 표시
