# UN 사무국 가평 유치 기사 시각화 기획안

## 페이지 구조 설계

### 1. Hero Section
**목적**: 충격적인 의제를 강렬하게 제시

```
┌─────────────────────────────────────┐
│  [타파스 뉴스레터 뱃지]               │
│                                     │
│  UN 사무국을 가평에?                 │
│  통일교 숙원 해결 앞장선 가평군수     │
│                                     │
│  현직 군수가 종교의 '숙원'을          │
│  공적 시스템으로 해결하려 한다        │
│                                     │
│  [날짜] [기자]                       │
└─────────────────────────────────────┘
```

**구성 요소**:
- 카테고리 뱃지: "타파스" (newsletter 스타일)
- 제목: 질문 형식으로 충격 전달
- 부제: 문제의 본질 짚기

```

│  🌍 UN 사무국 현황                   │
│                                     │
│  [세계 지도]                         │
│  ① 미국  ② 스위스  ③ 오스트리아      │

│                                     │
│  통일교의 목표:                      │
│  ⑤ 한국 가평 천원궁 (통일교 본부)    │
└─────────────────────────────────────┘
```

**인터랙션**:
- 지도에 핀 마크로 4곳 표시
- 5번째 희망 위치 강조 (빨간색)
